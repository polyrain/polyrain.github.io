Java application to generate Vim theme layouts using a JavaFX based GUI, submitted to the UQCS Hackathon in 2018. My contribution centred on GUI development.

 
Uno for the UNIX command line. Still in active development; using ncurses for a GUI and non-blocking IO for communication between clients to enable chat.

I was the president of the Design Society at UQ for 2020! Details on our projects, including a slackbot and our code-of-conduct, can be found here.

 
You can find the vimrc setup that I use to write most of work on my Github, linked in the photo here. I've found it super useful, and maybe you will too!
